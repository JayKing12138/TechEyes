
这个项目叫 **TechEyes**，意思很明确：它是用户观察科技行业的“眼睛”。这双眼睛不仅能看清现在，还能回溯历史，预测未来。

以下是用户进入 **TechEyes** 网页后的**保姆级功能描述**，以及后台是如何把那些大厂都在吹的“新技术”给实实在在用进去的。

---

## **TechEyes 产品功能与技术深度拆解**

### **1. 启动页：定标与意图捕捉**

**【用户操作】**：进入网页，在像 Google 一样的简洁搜索框里输入：“我想看低空经济（eVTOL）的发展，重点分析大疆和亿航，对比它们的历史路线，给我未来的投资建议。”
**【功能表现】**：

* **语义缓存 (Semantic Cache)**：系统先在后台“搜”一下。如果有别的用户刚问过类似问题，直接从缓存调取已经分析好的逻辑框架，不用重新花钱调大模型，速度极快。
* **上下文工程 (Context Engineering)**：系统不会直接把你的原话发给 AI，而是自动补全背景（比如现在的日期、行业基准知识），把你的话翻译成更专业的“分析指令”。

> **技术备注**：利用 **Redis** 做语义缓存；通过模板化的 **System Prompt** 进行上下文增强。

---

### **2. 过程页：看着 AI 怎么干活**

**【用户操作】**：点击开始后，页面中间会出现一个不断扩展的“任务地图”。
**【功能表现】**：

* **多智能体 (Multi-Agent) 与 思维链 (CoT)**：你会看到一个“主考官 Agent”把任务分给了三个“打工人”：一个去翻旧报纸（历史），一个去刷今天的新闻（现状），一个去算逻辑（未来）。
* **MCP 协议与 Function Calling**：那个刷新闻的 Agent 会自动“变出”一双手，通过 **MCP 协议** 直接连接到你的数据库或者 Google 搜索接口，自动调用工具。
* **A2A 协议 (Agent-to-Agent)**：如果系统发现需要画个对比图，它会通过 **A2A 协议** 发消息给另一个专门画图的 Agent 协作。

> **技术备注**：用 **LangGraph** 控制任务流；用 **Anthropic 的 MCP** 标准连接外部工具；**DeepSeek-R1** 输出思考过程。

---

### **3. 深度分析区：纠错与记忆**

**【用户操作】**：AI 搜到一个信息说“亿航某型号已量产”，你觉得不对，在旁边点一下“存疑”。
**【功能表现】**：

* **自纠错机制 (Self-Correction)**：Agent 收到你的质疑，立刻反思：“糟糕，我可能被营销号骗了。”它会启动 **Reflection 范式**，重新去找官网公告交叉验证。
* **短期与长期记忆 (Memory)**：你聊到一半断网了，刷新网页，AI 依然记得你刚才关注的重点（短期）。甚至你一个月后再登录，它还记得你偏好“技术细节”多于“财务数据”（长期）。
* **RAG (知识检索)**：它会把刚才搜到的几百篇文档存进 **向量库**，你问细节时，它能瞬间定位到哪篇文档的哪一页。

> **技术备注**：基于 **LangGraph 的 Checkpointer** 实现短期记忆；用 **Vector DB (ChromaDB)** 实现 RAG；编写 **Reflection Node** 实现自纠错。

---

### **4. 结果页：一份会动的报告**

**【用户操作】**：最后生成一份精美的网页报告，里面有时间轴、对比表。
**【功能表现】**：

* **结构化输出 (Structured Output)**：报告不是一堆乱七八糟的文字，而是规整的 JSON 转成的网页。
* **未来展望推演**：AI 会给出一个“逻辑推演路径”。你可以改动一个条件（比如：如果政府补贴取消了），AI 会基于 **思维链** 重新推导一遍结论。

> **技术备注**：使用 **Pydantic** 强制要求模型按格式输出；利用推理模型（Reasoning Model）的长逻辑链做趋势预测。

### **总结：TechEyes 的核心技术亮点（面试必背）**

| 功能 | 大白话解释 | 用到的硬核技术 |
| --- | --- | --- |
| **聪明地搜** | 动态去网上抓，不是死搜库 | **Agentic RAG + Function Calling** |
| **分工协作** | 好几个 AI 各司其职，还能互相发消息 | **Multi-Agent + A2A 协议** |
| **知错就改** | 发现信息打架了，自动去找证据核实 | **Self-Correction (Reflection)** |
| **过目不忘** | 记得你之前的所有要求和偏好 | **Memory (Short/Long-term)** |
| **逻辑严密** | 预测未来不是瞎猜，是有一套推导链条 | **CoT (思维链) + DeepSeek-R1** |
| **连接万物** | 能直接读你的本地文件或特定数据库 | **MCP 协议** |
